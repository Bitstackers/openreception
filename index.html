Welcome to the OpenReception Project

The OpenReception Project is a telephony-oriented software stack that provides a front desk reception user interface, along with a set of management tools that, among other things, provides a complete revision history of objects changed in the system.

OpenReception can be informally described as web-oriented glue around the software-PBX FreeSWITCH, is written in Dart and is currently in a version 2-rewrite Alpha state.

Updates can be followed on GitHub.
